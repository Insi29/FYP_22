About

Air writing is a concept to write in free space without any aid of stylus, graphical tablets etc. It is a revolutionized software especially built for visually impaired users, hyplexic people and people with Language Development Diorder. Allows users to write in front of webcam just by moving finger in air, as the software detects the finger using Media Pipe and creates writing event using Open CV by its movement. By combining different state of arts we also integrate recognition of words where users write and get recognized by the machine by applying ML models and OCR techniques using NanoNets. This recognized text is shown on screen to let users know that machines predicts what they write. Also our software provides feature to convert Text written in Air to be converted into audio/speech using 'pyttsx3' library. This software is further portrayed or accessed by website built using HTML, CSS and Django.
